# nabadOS System Manual

Welcome to the **sharafdin.com** engineering environment. This document explains the features and commands available on this terminal-driven platform.

## üìü The Terminal (nabad-sh)
The terminal is your primary interface for deep interaction with the site.
- **Toggle**: Press `` ` `` (backtick) or click the floating **TERM** button.
- **Persistence**: Your username and theme are saved locally.
- **Auto-completion**: Press **Tab** to complete commands or see suggestions.

### Command Reference
| Command | Usage | Description |
| :--- | :--- | :--- |
| `help` | `help` | List available commands. |
| `ls` | `ls` | List site pages and posts. |
| `cat` | `cat [file]` | Navigate to a page or read a file (e.g., `cat blog`). |
| `tree` | `tree` | Display the site's directory structure. |
| `demo` | `demo` | Start an automated tour of the terminal features. |
| `scan` | `scan` | Run a simulated security audit on the environment. |
| `scanline` | `scanline` | Toggle the "Live Scan" background overlay. |
| `neofetch` | `neofetch` | Show system info and nabadOS logo. |
| `projects` | `projects` | List major engineering projects (Soplang, nabadOS). |
| `stats` | `stats` | View real-time (mock) system resource telemetry. |
| `history` | `history` | Show a log of commands run in this session. |
| `theme` | `theme [name]`| Change terminal color (classic, amber, matrix, cyberpunk, brutalist). |
| `setname` | `setname [name]`| Update your terminal username. |
| `clear` | `clear` | Clear the terminal output window. |
| `whoami` | `whoami` | Print your current username. |
| `exit` | `exit` | Close the terminal interface. |

## üõ∞Ô∏è System Features
The environment includes several advanced visual and technical modules:

### 1. System Status HUD
Located in the bottom-left corner, the HUD provides persistent telemetry:
- **CPU**: Real-time load fluctuation.
- **NET**: Connection status (Always ENCRYPTED).
- **LOCALE**: Current site language (EN_US or SO_639).

### 2. Visual Modules
- **CRT Overlay**: A subtle flicker and scanline effect to mimic 80s hardware.
- **Glitch Headers**: Hovering over section titles (e.g., "RESEARCH") triggers a data-corruption animation.
- **Background Grain**: A tactile texture applied across the dark-mode theme.

### 3. Heritage Integration
- **Somali Star Map**: Found in the **Science** section. An ASCII representation of constellations with names in Somali (toggle via EN/SO buttons).
- **Full i18n**: Real-time translation of all major UI elements and selected blog posts (e.g., "Welcome").
